Project Overview

Helping Hands NGO is a PHP-based nonprofit website designed to manage campaigns, accept donations, organize volunteers, and present events in a polished public interface.

The project includes a fully structured front-end with PHP templates, backend database files, and documentation so the app can be deployed quickly in a local or hosted PHP/MySQL environment.

  • Donation management and receipt tracking
  • Volunteer signup and profile handling
  • Event listings with schedule details
  • Reports and documentation generation
Note: GitHub Pages shows this static landing page only. The full PHP site runs on a web server with PHP and MySQL support.

How to run the full site

Use a local PHP/MariaDB environment to preview the full dynamic site and database features:

1. Install XAMPP or WAMP.
2. Place the repository folder inside your server document root.
3. Import database.sql into MySQL.
4. Open http://localhost/Helping Hands NGO.php/ in a browser.
5. Use index.php to access the website.
Tip: This `index.html` is best for a public GitHub Pages preview and for guiding reviewers to the right project files.
